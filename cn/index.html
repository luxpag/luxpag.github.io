# 简介

      Luxpag 根植于拉美，以互联网支付、移动支付为主要业务；面向商户，为商户提供快捷、安全的支付通道，为商户的客户提供便捷、安全的支付系统。

      Luxpag 支付可对接国家：巴西、墨西哥、哥伦比亚、智利、秘鲁、厄瓜多尔、巴拿马、萨尔瓦多、哥斯达黎加、危地马拉、尼加拉瓜等国家。

# 开发指南

      Luxpag 支付分为两类：1、集成支付（推荐），2、直连支付。

      1、集成支付：商户调用 Luxpag 创建订单接口（商户开发工作），传订单基本信息和商户信息；Luxpag 创建订单后，返回“支付地址”（ Luxpag 支付页面，如图1）；商户跳转到 Luxpag 支付页面，用户在支付页面填写支付信息提交订单付款。

      Luxpag 收到支付结果后，回调（IPN通知）商户系统（商户开发工作）。

（图1）

      2、直连支付：商户调用 Luxpag 直连支付接口，传商户信息、用户信息、订单信息（商户需自己开发支付页面），Luxpag 收到信息后生成订单，同时返回订单结果给商户（商户需开发支付成功页面）。

      Luxpag 收到支付结果后，回调（IPN通知）商户系统（商户开发工作）。
