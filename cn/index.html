# 简介

      Luxpag 根植于拉美，以互联网支付、移动支付为主要业务；面向商户，为商户提供快捷、安全的支付通道，为商户的客户提供便捷、安全的支付系统。

      Luxpag 支付可对接国家：巴西、墨西哥、哥伦比亚、智利、秘鲁、厄瓜多尔、巴拿马、萨尔瓦多、哥斯达黎加、危地马拉、尼加拉瓜等国家。

# 开发指南

      Luxpag 支付分为两类：1、集成支付（推荐），2、直连支付。

      1、集成支付：商户调用 Luxpag 创建订单接口（商户开发工作），创建订单接口会返回“支付地址”（ Luxpag 支付页面，如图1）；商户需跳转到 Luxpag 支付页面，用户填写支付信息提交订单并付款（如PIX，可扫码付款，如图2）。

      Luxpag 收到支付结果后，回调（IPN通知）商户系统（商户开发工作）。

      2、直连支付：商户调用 Luxpag 直连支付接口，Luxpag 收到信息后生成订单，同时返回创建订单结果给商户（商户需开发支付成功页面）（如PIX，可自己显支付二维码信息或使用luxpag checkout页面，如图2）。

      Luxpag 收到支付结果后，回调（IPN通知）商户系统（商户开发工作）。
